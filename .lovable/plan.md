

# Landing Page FacioFlow

## Visão Geral
Landing page moderna e profissional para a FacioFlow, empresa de soluções tecnológicas e automações com IA. Design baseado em tons de azul e preto, fonte Poppins, componentes Shadcn e botões de CTA com links placeholder prontos para edição.

---

## Design & Estilo
- **Fonte**: Poppins (Google Fonts)
- **Cores**: Azul (#1E40AF / #3B82F6) como primária, preto/dark como fundo de seções alternadas, branco para contraste
- **Logo**: Logo branca da FacioFlow (enviada pelo usuário) usada no header e footer sobre fundo escuro
- **Ícones**: Lucide React (já instalado, compatível com Shadcn)
- **Layout**: Responsivo, mobile-first

---

## Estrutura das Seções

### 1. Header/Navbar
- Logo FacioFlow à esquerda
- Links de navegação para as seções (Soluções, Segurança, Como Trabalhamos)
- Botão CTA "Fale Conosco" com link placeholder

### 2. Hero Section
- Fundo escuro (preto/azul escuro) com gradiente sutil
- **H1**: "Inteligência tecnológica que escala seu negócio"
- **Parágrafo**: Texto descritivo sobre soluções e automações com IA
- Botão CTA grande com link placeholder

### 3. Destaques (4 cards)
- Cards com ícones para: Desenvolvimento orientado ao produto, Automação de Processos, Implementação rápida, Segurança de Dados
- Layout em grid 2x2 (desktop) / empilhado (mobile)

### 4. Soluções (Seção principal)
- **H2**: "Criamos para solucionar o seu problema"
- Parágrafo introdutório
- 7 blocos de serviços, cada um com ícone, título H3 e descrição:
  - Automação de Processos
  - Criação de Dashboards de BI
  - Desenvolvimento de Soluções Sob Medida
  - Desenvolvimento de Aplicativos Web e Mobile
  - Desenvolvimento de Chatbots Conversacionais
  - Agentes de IA e Sistemas RAG
  - Análise de Métricas Sociais por IA
- Layout em cards/grid alternado para visual dinâmico

### 5. Segurança e Privacidade
- Fundo escuro para destaque
- **H2**: "Segurança e Privacidade de Dados"
- Texto sobre LGPD/GDPR com ícones de segurança (Shield, Lock)
- Visual que transmita confiança

### 6. Como Trabalhamos (Timeline/Steps)
- **H2**: "Como Trabalhamos"
- 5 etapas em formato de timeline/stepper visual:
  1. Análise
  2. Especificação
  3. Construção
  4. Lançamento
  5. Otimização Contínua
- Cada etapa com número, título e breve descrição

### 7. CTA Final
- Fundo com gradiente azul
- **H3**: "Pronto para elevar os processos da sua empresa?"
- Parágrafo motivacional
- Botão CTA grande com link placeholder

### 8. Footer
- Logo FacioFlow
- Links de navegação
- Links de redes sociais (placeholder)
- Copyright

## Botões CTA
Todos os botões terão links placeholder (`href="#"`) com comentários no código para fácil localização e substituição posterior.
